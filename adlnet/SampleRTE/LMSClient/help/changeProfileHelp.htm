User Preferences Help
User Preferences Administration allows a user to edit or change his or her password and set additional student preferences, which include audio_level, language, delivery_speed and audio_ captioning.

Edit User Profile Instructions
  1. The current password displayed in the password text field can be modified and or changed by selecting the text field and modifying as needed. Upon selecting the submit button the new password will be changed to the updated text within the password text box or kept the same if no modifications were made.
  2. To modify the learner_preference.audio_level select the audio_level preference text box and enter in a real number 0..*.
  3. To update the cmi.learner_preference.language select the language text box and enter a valid language code.
  4. To update the cmi.learner_preference.delivery_speed select the delivery speed text box and enter a valid real number 0..*.
  5. To update the cmi.learner_preference.audio_captioning select the audio_captioning text box and enter a valid choice.
    • -1 represents the "off" state
    • 0 represents "no change" state
    • 1 represents the "on" state
  6. Click the Submit button.
