Launch Course Help
When viewing the list of courses that you are registered for, you also have the ability to launch these courses. In order to launch a course, you first must have registered for this course. If you need help registering for a course, you can find it here.

Launch Course Instructions

If you would like to begin a new attempt on a course, please select Start Course.

    Note: This option will not be available if you are required to
    choose an activity to begin the course; in these cases please
    select View Table of Contents.

If you suspended your previous attempt on a course, please select Resume Course to continue that attempt.

If you would like to choose an activity to begin a course, please select View Table of Contents to see which activities are available. After viewing the Table of Contents, please choose an activity to begin the course.

    Note: This option will not be available if the course does
    not provide a Table of Contents; in these cases please select
    Start Course or Resume Course.
