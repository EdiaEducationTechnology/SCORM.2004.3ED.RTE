Global Objectives Administration Help
Global Objectives allow progress information for a student to be shared by more than one SCO. For more information on Global Objectives, refer to SCORM 2004 3rd Edition.

Global Objective Administration Instructions
  1. New Global Objectives may be created or existing Global Objectives may be modified.
  2. To create a new global objective, select Create Global Objectives, fill in the necessary information, and click Submit.
  3. To view or modify an existing global objective, click on View or modify an existing global objective. Choose the user whose global objectives you would like to view or modify, and click Submit.
  4. Select the course whose Global Objectives you would like to view. All objectives not associate with a course may also be viewed. If you choose a course and no global objectives are associated with that course, all global objectives which are not associated with any course will be displayed.
  5. Click the Submit button.
  6. To modify global objectives, select the objectives to delete or reset individually or use the Select All option to select all objectives. The Clear Selection option allows deselection of objectives that have been selected.
  7. Click the Submit button.
