Course Import Help

Course Import allows an administrator to import a SCORM® Conformant Content Aggregation Package into the Sample Run-Time Environment.

Import Instructions
  1. Click the Browse button and select the Zip file that contains the course. This Zip file MUST BE a valid SCORM 2004 3rd Edition Content Aggregation Package.
  2. Select the radio button indicating whether or not you would like to validate the course.
    NOTE: If you choose to not validate the course and it contains errors, it may behave unexpectedly.
