Installation

It is recommended that you read through all of the Installation Instructions prior to beginning installation.

Prerequisites

This release of the Sample RTE requires the Sun Java 2 Software Development Kit, Standard Edition Version 1.4.2 or above. The Sun Java 2 Software Development Kit, Standard Edition is available for download free of charge at http://java.sun.com/products/archive/ At the time of release of this software, the latest version of the Sun Java 2 Software Development Kit, Standard Edition is 1.5.0_9.

Notes:
  • The Sample RTE requires the full J2SE SDK. The J2SE JRE alone is NOT sufficient for this software.
  • The Sample RTE client may be accessed from a machine other than the one on which it is installed using Microsoft Internet Explorer. In this case, the J2SE Runtime Environment (JRE) must be installed on the second machine.

After installing the J2SE SDK, you must do the following:

  1. Create an environment variable named "JAVA_HOME" and set its value to the root directory of the J2SE SDK.
    • For Windows 2000 and XP: Right click on "My Computer", select "Properties", select the "Advanced" tab and then click on "Environment Variables". Add a JAVA_HOME entry in the User or System Variables that points to the location where the J2SE SDK was installed (i.e., C:\j2sdk1.4.2_05).
  2. Add the full path of the J2SE SDK bin directory (where the java.exe is located) to the PATH environment variable.
    • For Windows 2000 and XP: Right click on "My Computer", select "Properties", select the "Advanced" tab and then click on "Environment Variables". Add the J2SE SDK bin directory to the Path by adding a semicolon followed by a reference to the JAVA_HOME environment variable followed by "\bin" (i.e., %JAVA_HOME%\bin).

Constraints

As of this release, the Sample Run-Time Environment has been tested on Microsoft Windows 2000 Professional (SP4) and Windows XP Professional SP2, both on the Intel Pentium PC platform with Microsoft Internet Explorer 6.0.

The Sample RTE has not been formally tested with any other browsers on any other platforms than those indicated above.

Installation

Installation of the Sample Run-Time Environment (RTE) is to be done only after installing the J2SE SDK as described in the Prerequisites.

The Sample RTE is available for download as a self-extracting executable install file. Once you have completed downloading the file, perform the following steps:

  1. Double-click on the downloaded self-extracting executable file to begin the installation process.
  2. Click the "Next" button to continue from the "Welcome" page.
  3. Read the "License Agreement" and
    • Click the "Yes" button to agree and continue with the install.
    • Click the "No" button if you do not agree with the terms of the license and wish to exit the install.
  4. The recommended root installation directory is "C:\ADL". You may choose another root installation directory, but the Sample RTE install will always append the "\ADL\SCORM_2004_3rd_Ed_Sample_RTE_V1.0" sub-directory to the specified root installation directory. The remainder of this Readme document assumes that the Sample RTE will be installed to "C:\ADL\C:\ADL\SCORM_2004_3rd_Ed_Sample_RTE_V1.0". This is referred to as "<drive>:\<install_directory>". If you override the default to use a different root installation directory, please take that into consideration when reading the remainder of this Readme.
  5. Click the "Next" button
  6. The install will add program icons to the Program Folder of your Start menu. You may type a new folder name, or select one from the existing list of Start Menu folders. It is recommended that you use "ADL". Then click on the "Next" button to install the Sample RTE. The installation program will copy the necessary files to the appropriate directories at this time.
  7. You will have the option to view a copy of this Readme document.
    • Check the "Show the Readme file" checkbox to view the Readme.
    • Uncheck the "Show the Readme file" checkbox to skip viewing the Readme.
  8. Click on the "Finish" button to complete the installation.
NOTE: The installation will not remove/un-install the Sample Run-Time Environment Version 1.3.3.

NOTE: The jakarta-tomcat-5.0.16 Servlet Engine and Web server will be installed and pre-configured to bind to port 8080.

Configuration

After installation, the following actions are necessary to ensure that the Sample RTE will run properly:
  • The Microsoft Internet Explorer browser must be set to check for a new page every visit to the page. To configure the browser:
    • Open a Microsoft Internet Explorer browser.
    • Click on "Tools" -> "Internet Options".
    • On the "General" tab, "Temporary Internet files" sub-section, click "Settings".
    • Ensure that the "Every visit to the page" radio button is selected.
    • Click "OK" and then "OK" again.

  • The Java Plug-in must be configured to disable caching. To disable the Java Plug-in cache:
    • For Java versions up to 1.5.0
      • Open the Windows "Control Panel".
      • Double click on the "Java Plug-in" icon.
      • Select the "Cache" tab.
      • Un-check the "Enable Caching" option.
      • Click "Apply".
      • Close Window


    • For Java versions after 1.5.0
      • Open the Windows "Control Panel".
      • Double click on the "Java Plug-in" icon.
      • Select the "General" tab.
      • Under the "Temporary Internet Files" section, click the "Settings" button.
      • In the "Temporary Files Settings" dialog box, click the "View Applets... " button.
      • In the "Java Applet Cache Viewer" dialog box, ensure the "Enable Caching" check box is unchecked for the Sample Run-Time Environment Applet(s).

  • Ensure that the Sun Java Plug-in is selected as the default Java Environment for the Microsoft Internet Explorer browser. To set the default browser Java Environment:
    • Open the "Java Plug-in" applet in the Control Panel
    • Select the "Browser" tab
    • Select the checkbox next to the browser that you are using to run the Sample RTE
    • Click "Apply"

  • If you are using Windows XP SP2 these additional actions are required:
    • Allow JavaScript to run by:
      • Open a Microsoft Internet Explorer browser
      • Click on "Tools" -> "Internet Options"
      • On the "Advanced" tab, check the box that states, "Allow active content to run in files on My Computer" under the "Security" section.
    • Allow the Sample RTE to pop-up windows by either disabling pop-up blocking or exclude the Sample RTE from pop-up blocking.

This completes the installation of the Sample RTE.
