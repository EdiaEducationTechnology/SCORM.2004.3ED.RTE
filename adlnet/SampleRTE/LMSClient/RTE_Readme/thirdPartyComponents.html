    The Table of Contents Menu is used when the Sequencing Control Choice mode is enabled. The menu is presented in the left Sample RTE frame.

      The Apache Xerces XML Parser is used to validate the XML in the imsmanifest.xml file during course import.

    • Apache Ant Version 1.6.2 (ant.jar)
      The Apache Ant ant.jar provides access to the Apache classes ZipFile and ZipEntry. The ZipFile class allows the encoding of an incomming zip file to be specified which enables support of certain foreign characters in filenames during import.
